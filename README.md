### How to play

1. Clone this repo

```bash
git clone https://github.com/kunish/minesweeper.git
```

2. Install dependencies

```bash
yarn
```

or

```bash
npm i
```

3. Start a local service

```bash
yarn start
```

or

```bash
npm run start
```

4. Game time!!!
